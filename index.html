Data Sharing System

The Data Sharing System, our capstone project at Eastwoods Professional College of Science and Technology, aims to improve efficiency by replacing outdated paper-based data processing with a streamlined, web-based solution for managing data across multiple departments.

System Features:

Record Viewer: Centralized access to student records.

ID Creation: Facilitates ID creation for SASO.

Confidential Data Entry: Allows the Guidance Department to enter data confidentially.

Remarks Feature: Enables the Clinic to track students' conditions.

Tuition Fee Management: Helps the Finance Department manage students' tuition fees.
